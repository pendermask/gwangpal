최후의 문법 속성과 img






태그와 속성

<img src="(이미지 링크)">
여기서 src는 source의 줄임말로 속성(attribute)입니다.
<img> 태그만으로는 원하는 이미지를 불러올 수 없기 때문에 속성이 붙은 것입니다.
이미지는 인터넷에 있는 이미지의 주소를 가져오거나 컴퓨터에 있는 이미지를 가져올 수도 있습니다.
            1. 인터넷에서 원하는 이미지의 주소를 홈페이지에 넣고 싶을 때
            <img src="https://ko.wikipedia.org/wiki/PNG#/media/%ED%8C%8C%EC%9D%BC:PNG_transparency_demonstration_1.png">
            
            

            2. 컴퓨터에 있는 이미지를 홈페이지에 넣고 싶을 때
            <img src="../IMG/구독뱃지 12.png">

* 컴퓨터에 있는 이미지를 가져올 때, 따로 이미지만 모아넣는 파일을 만들고 따로 저장한다면 경로를 쉽게 적을 수 있습니다.

              <img src="../IMG/10.PNG">
위 이미지는 현재 HTML을 다루고 있는 폴더 이미지로, 방송 폴더에 있던 그림 중에 원하는 이미지만 골라와 따로 저장해주었습니다.
..는 상위 폴더를 뜻하며 /로 이동을 합니다. 그리고 원하는 파일의 이름을 적어주시면 됩니다.
여기서 속성 명칭을 실수한다면 홈페이지에 이미지가 안 올라갈 수 있으니 주의하시길 바랍니다.
ex) ㅁㄴㅇㄹ.png를 ㅁㄴㅇㄹ.PNG로 적게 되면 화면에 뜨지 않을 수 있습니다.

** 무료 이미지 다운로드 홈페이지 링크

이미지 크기

이미지의 크기를 조절할 수 있습니다.
추천 검색어 : html img size attribute

'width' 속성의 값으로 숫자나 %를 사용하여 크기를 조정합니다.
비교를 위해 이미지의 원본과 width 속성을 이용하여 크기를 바꿔보겠습니다.

원본

width 속성 100px

width 속성이 사용된 코드는 다음과 같습니다.
<img src="../IMG/구독뱃지 12.png" width="100px">


* 이미지의 크기를 바꿔주는 속성은 width 뿐만 있지 않으니, 궁금하시다면 html img size attribute 를 검색해보세요.
